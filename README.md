分支 M0 # 最新版本 M0.0.1 支持的功能：

1. 将两个字幕文件按序号合并。

2. 可以选择将字幕 1 的内容显示在合成字幕中的上方还是下方。

3. 可以选择将程序中的字幕读取过程输出至屏幕或者本地文本。

4. 统计程序运行用时。

---

分支 M1 # 最新版本 M1.0.0 支持的功能：

1. 可以批量纠正 .srt 文件字幕的编号，使其按在文本中出现的顺序，以正整数递增为序号重新编号。
